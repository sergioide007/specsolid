# SpecSolid | Alpaquitay AI 🚀

## Overview
**SpecSolid** is an open-source initiative dedicated to the research and development of robust, scalable, and decoupled software architectures. Our core project, **Alpaquitay AI**, focuses on bridging the gap between advanced Artificial Intelligence and practical, explainable implementation (XAI).

This project is maintained as a **personal research and community contribution** effort, aiming to share best practices in software engineering and AI orchestration.

## 🛠 Tech Stack & Principles
- **Architecture:** Decoupled RAG (Retrieval Augmented Generation), Hexagonal Architecture, and Microservices.
- **AI Focus:** Explainable AI (XAI) using SHAP/LIME, Multi-agent orchestration (n8n/LangChain).
- **Core Values:** "Principles over trends." We follow the Agile Manifesto and the RUN LIDER framework for technical excellence.

## 🌟 Alpaquitay AI
Alpaquitay AI is our flagship research model designed for:
- **Transparency:** Providing human-readable explanations for AI decisions.
- **Scalability:** Built on a decoupled Control Plane for seamless integration.
- **Identity:** Developed with a global perspective from Pucallpa, Peru.
